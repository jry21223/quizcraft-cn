# 题库解析功能使用指南

## 📋 新格式说明

### 题目格式对比

**原格式（无解析）：**
```
1、题目内容（ ）。
A、 选项A
B、 选项B  
C、 选项C
D、 选项D
答案：A
```

**新格式（带解析）：**
```
1、题目内容（ ）。
A、 选项A
B、 选项B
C、 选项C
D、 选项D
答案：A
解析：这里是详细的答案解析，解释为什么选这个答案，相关知识点说明等。
```

### 2. 后端修改（web_app.py）

修改 `build_question_list` 函数，解析解析字段：

```python
def build_question_list(contents: Dict) -> List[Question]:
    questions: List[Question] = []
    for chapter_name, chapter_data in contents.items():
        for item_type, items in chapter_data.items():
            for idx, item in enumerate(items):
                # 解析答案和解析
                if "答案：" in item:
                    split_idx = item.find("答案：")
                    stem = item[:split_idx]
                    rest = item[split_idx + 3:].strip()
                    
                    # 检查是否有解析
                    if "解析：" in rest:
                        analysis_idx = rest.find("解析：")
                        answer = rest[:analysis_idx].strip()
                        analysis = rest[analysis_idx + 3:].strip()
                    else:
                        answer = rest
                        analysis = ""
                else:
                    stem = item
                    answer = ""
                    analysis = ""
                
                qid = f"{chapter_name}|{item_type}|{idx}"
                questions.append(
                    Question(
                        id=qid,
                        stem=stem,
                        answer=answer,
                        analysis=analysis,  # 新增字段
                        chapter=chapter_name,
                        type=item_type
                    )
                )
    return questions
```

修改 `Question` dataclass：

```python
@dataclass
class Question:
    id: str
    stem: str
    answer: str
    analysis: str  # 新增字段
    chapter: str
    type: str
```

## 📚 批量添加解析

可以使用脚本批量为现有题库添加空解析字段：

```python
import json

with open('xigai.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

for chapter, types in data.items():
    for qtype, questions in types.items():
        for i, q in enumerate(questions):
            if '解析：' not in q:
                # 在答案后添加空解析占位符
                questions[i] = q + '\n解析：（待补充）'

with open('xigai_with_placeholder.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)
```

然后手动填写解析内容。
